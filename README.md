# python-web-app
Blood Bank and Donation System

**Overview**
This project is a web-based Blood Bank and Donation System designed to improve the management of blood donations and availability. It focuses on providing a centralized platform for hospitals, donors, and patients to interact in real time.

**Problem**
In many cases, the lack of a centralized system makes it difficult to quickly find available blood donors, leading to delays in emergency situations.

**Solution**
This system centralizes donor information and blood inventory into a single platform, allowing faster access to critical data and improving coordination between users.

**Features**
Donor registration and profile management
Blood inventory tracking
Real-time access to donor and blood type data
Admin dashboard for system management
Chatbot for answering common blood donation questions

**Technologies Used**
Backend: Python (Django)
Frontend: HTML, CSS, JavaScript
Database: SQLite
Additional: JavaScript-based chatbot

How to Run

# Navigate into the project
cd blood-bank-system

# Run migrations
python manage.py migrate

# Start the server
python manage.py runserver
Then open:
http://127.0.0.1:8000/


**Key Outcomes**
Faster identification of available donors
Improved response time in emergencies
Better organization and transparency of blood data

Building full-stack web applications using Django

Implementing user roles and admin panels
